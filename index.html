PROJECTS

Initial P

Computer System Architecture

My curiosity in cluster, cloud, and parallel computing has led me to wanting a practical implementation of multithreading and network programming. Initial P is an accumulation of my knowledge in programming languages, software design, network programming, and basic circuitry.

This project integrates the controller of a RC car to a Raspberry Pi. The RC car is controlled using an android application and/or computer keyboard inputs. Controller inputs are sent via wifi connection to the Raspberry Pi.

DC2S

Cloud Computing & Security

Distributed Cloud Computing with StarCluster (DC2S) utilizes Amazon's EC2 with StarCluster to simulate computational offloading for an android application.

Conceptually, an android application requiring more computional power or resources can delegate such task to a cloud server. The cloud server equiped with multiple computers will perform the compute-intensive task for the android application. Cloud or edge computing can be used as way to deliver computing power to mobile phones and IoT devices.

IMDb Recommendation System

Intro to High Performance Computing

A project involving parallel computation and data science. This program was designed to implement the k-means clustering algorithm using CUDA primitives—Nvidia's parallel computing platform and programming model.

It can be used as a movie recommendation system that separates user ratings and votes into groups. Ideally, a movie with high/low rating and votes will be assigned to a group. A group of movies that have a high rating and high number of votes can be recommended to users.
